About React

React is a library, not a framework.

It handles rendering and state, but leaves other things like data fetching and routing up to you.

Fortunately, thanks to the huge community, there are always many options to choose from.

History of React

  • Created in 2011 at Facebook
  • Made open source in 2013
  • React Hooks introduced in 2019
  • Current version is React 17
