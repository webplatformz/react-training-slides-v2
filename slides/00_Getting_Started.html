MacOS: Setup

This guide shall help you setup:
Node.js and NPM on MacOS.

How To

  1. Download the latest Node.js LTS version
  2. Open a shell (CMD + Space, Type: "Terminal")
  3. Type command: node -v
  4. Type command: npm -v

You have installed Node.js & NPM,
if the output looks similar.

Windows: Setup

This guide shall help you setup:
Node.js and NPM on Windows.

How To

  1. Download the latest Node.js LTS version
  2. Open a PowerShell
  3. Type command: node -v
  4. Type command: npm -v

You have installed Node.js & NPM,
if the output looks similar.

Common Problems

  • You already have some version of Node.js and NPM installed on your machine. Try to remove and start from scratch.
  • (Windows): The command node -v is not found. Even though you have installed it. Make sure that the path to your Node.js installation folder is set as an environment variable. If not, add it to the PATH variable.

Advanced: Setup

This section provides a few tips on how to setup Node.js and NPM using a package manager. The advantage of a package manager is that multiple Node.js versions can be installed in parallel, without conflicts.

Note! You don't have to install Node.js using a package manager. Using the previous setup guides is fine.

How To

  1. Remove all current installations of Node.js and NPM .
  2. Install the Package Manager
    • Windows: NVM-Windows (installation guide)
    • MacOS: NVM (installation guide)
  3. Install Node.js : nvm install lts
  4. Verify that everything has been installed correctly.
    See previous slides.
