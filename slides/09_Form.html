Different Form Tags

  • input / textarea
  • select
  • checkbox
  • radio

input / textarea

              
                <input
                    value={value}
                    onChange={e => { setValue(e.target.value) }}
                />

select

              
                <select
                  value={value}
                  onChange={e => { setValue(e.target.value) }}
                >
                  <option value="grapefruit">Grapefruit</option>
                  <option value="lime">Lime</option>
                </select>

checkbox

              
                <input
                  type="checkbox"
                  checked={checked}
                  onChange={(e) => { setChecked(e.target.checked) }} />

radio

              
                <input
                  type="radio"
                  name="preference"
                  value="tomato"
                  checked={option === "tomato"}
                  onChange={(e) => { setOption(e.target.value) }}
                />

Controlled vs. Uncontrolled

Controlled

Update state with each change event

                
                    <input
                      value={value}
                      onChange={e => { setValue(e.target.value) }}
                    />

Uncontrolled

Read values only on blur / submit to prevent needless re-renders.

                
                    <input
                      defaultValue={value}
                    />
To leverage uncontrolled form fields, using a library is advised

Form Libraries

Form handling can be quite repetitive, the usage of a helper library should therefore be considered.

  • React Hook Form
  • Formik

React Hook Form

              
                const {
                  register,
                  handleSubmit,
                } = useForm();
              
            
              
                <form onSubmit={handleSubmit(onSubmit)}>
                  <input defaultValue="test" {...register("example")} />
                  <input type="submit" />
                </form>
